Conda

2022-01-12 deeplearning
创建环境

conda create --name py3.7 python=3.7

激活环境

conda activate py3.7

退出环境

conda deactivate

删除环境

conda remove --name py3.7 --all

查看所有环境

conda env list

conda安装numpy模块

conda install numpy

查看安装的模块

conda list
或者
conda list --name py3.7

查看某个模块的信息

conda search numpy

删除模块

conda remove numpy
或者
conda remove --name py3.7 numpy

更新模块

conda update numpy

或者
conda update --name py3.7 numpy

更新全部模块：
conda update --all

更新conda

conda update conda

更新python

conda update python

生成.condarc文件

conda config --set show_channel_urls yes

设置channels

设置国内源，这里北京外国语的conda源，或者也可以就搞个魔法工具，用默认的源

这俩是必须有的：
conda config --add channels https://mirrors.bfsu.edu.cn/anaconda/pkgs/main/

conda config --add channels https://mirrors.bfsu.edu.cn/anaconda/ pkgs/free/

安装pytorch，你需要这个：
conda config --add channels https://mirrors.bfsu.edu.cn/anaconda/cloud/pytorch/

最重要的一点就是，要加上pytorch/
网上很多教程都是到cloud/就结束了

安装pytorch的时候，在官网复制的，请去掉最后的-c pytorch，这个命令是去pytorch官网下载pytorch系列模块

查看channels

conda config --show channels

删除channels

conda config --remove channels <urls>

删除所有channels

conda config --remove-key --channels

恢复默认channels

conda config --remove-key channels
